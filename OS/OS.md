## 운영체제란?

- Operating System 또는 OS 라고 부름
- 대표적인 운영체제
  - Windows OS, Mac OS, 그리고 UNIX(유닉스) 
  - UNIX OS
    - UNIX 계열 OS
      - UNIX와 사용법이나, OS 구조가 유사
    - LINUX (리눅스) OS
      - 프로그래머, 전공자
